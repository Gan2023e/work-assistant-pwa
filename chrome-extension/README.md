# 产品审核助手浏览器插件

## 功能概述

这是一个专为采购链接管理系统设计的产品审核助手浏览器插件，能够自动获取产品网页源代码并保存到后端系统。

## 主要功能

### 🆕 新版产品审核流程（推荐）

**分步审核模式** - 更安全、更可控的审核体验：

1. **选择产品** - 在采购链接管理页面勾选要审核的产品
2. **开始审核** - 点击"新品审核"按钮，插件会先审核第一个产品
3. **逐个审核** - 每个产品审核完成后，点击"下一个产品"继续
4. **随时停止** - 可以随时点击"完成审核"停止审核流程
5. **审核总结** - 所有产品审核完成后显示详细总结

**优势：**
- ✅ 更安全：一次只处理一个产品，避免浏览器卡顿
- ✅ 更可控：可以随时停止，支持断点续审
- ✅ 更友好：实时显示进度，清晰的操作提示
- ✅ 更稳定：减少内存占用，提高成功率

### 🔄 旧版批量审核流程

**批量审核模式** - 一次性处理所有选中的产品：

1. 选择多个产品
2. 点击"新品审核"按钮
3. 插件自动批量处理所有产品
4. 完成后显示结果

## 安装说明

### 方法1：开发者模式安装（推荐）

1. 打开Chrome浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择插件文件夹
5. 插件安装完成

### 方法2：打包安装

1. 在开发者模式下点击"打包扩展程序"
2. 选择插件文件夹，生成.crx文件
3. 将.crx文件拖拽到扩展程序页面安装

## 使用方法

### 1. 准备工作

- 确保已登录采购链接管理系统
- 确保有足够的权限进行产品审核
- 确保网络连接正常

### 2. 开始审核

1. 进入采购链接管理页面
2. 勾选要审核的产品（支持多选）
3. 点击页面上的"新品审核"按钮
4. 确认审核信息后开始审核

### 3. 审核流程

#### 新版分步审核：
- 第一个产品审核完成后，会显示结果弹窗
- 点击"下一个产品"继续审核下一个
- 点击"完成审核"结束当前审核流程
- 所有产品完成后显示审核总结

#### 旧版批量审核：
- 插件会自动处理所有选中的产品
- 处理过程中会显示进度提示
- 完成后显示批量结果

### 4. 结果查看

- 每个产品的审核结果都会实时显示
- 可以查看获取的网页源代码
- 支持复制源代码到剪贴板
- 失败的产品会显示错误信息

## 故障排除

### 常见问题

1. **插件不显示**
   - 检查是否在正确的页面
   - 确认插件已启用
   - 刷新页面重试

2. **审核失败**
   - 检查登录状态
   - 确认网络连接
   - 查看控制台错误信息

3. **页面卡顿**
   - 减少同时审核的产品数量
   - 使用分步审核模式
   - 关闭其他不必要的标签页

### 调试模式

1. 打开浏览器开发者工具
2. 查看Console标签页的日志信息
3. 检查Network标签页的网络请求
4. 查看插件背景页的日志
